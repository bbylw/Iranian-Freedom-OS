## English (EN)

An interactive web-based operating system simulator and digital art project. It vividly demonstrates the transition from a totalitarian "National Information Network (SHOMA)" to a free, secular, and democratic regime.

### 📖 Project Brief

Initially presented as **Melli OS** (سیستم‌عامل ملی) — a fictional operating system inspired by the totalitarian censorship and surveillance of the Islamic Republic of Iran. In this phase, users experience network disconnection, surveillance by FATA (Cyber Police), mandatory hijab decrees, and an ubiquitous sense of oppression.

However, by executing a specific "Liberation Protocol" command in the **Terminal**, users can breach the SHOMA firewall, transforming the system into a symbol of light and freedom. The liberated system is centered around the slogan "**Woman, Life, Freedom**" (زن، زندگی، آزادی), exposing historical truths and presenting the _Secular Democratic Manifesto_.

### ✨ Key Features

- 💻 **Immersive Desktop**: Features boot animations, login interface, Dock, and a draggable skeuomorphic window system.
- 🌍 **Multilingual & RTL Support**: Native support for **Persian (FA)**, **English (EN)**, and **Chinese (ZH)**. Perfectly adapted for Right-to-Left (RTL) layout with the open-source Vazirmatn font.
- 👁️ **FATA Monitor Simulation**: Real-time simulation of totalitarian network monitoring, packet interception, keyword censorship, and CPU load monitoring.
- ⌨️ **Interactive Terminal**: Includes functional command-line interactions (e.g., `help`, `ls`, `cat`, `whoami`, `history`).
- 🔓 **"Liberation" Protocol**: Triggered via hidden commands with full-screen transition animations, switching the theme from oppressive dark green to the "Tricolour" dawn (Green, White, Red).
- 📜 **Dynamic Archives**:
  - **Totalitarian State**: Displays "Revolutionary Guards Cyber Directives".
  - **Liberated State**: Unlocks the "Woman, Life, Freedom - Secular Democratic Manifesto" and the "Indictment of the Religious Dictatorship".

### 🚀 How to Use

1. **Run Project**: Open `index.html` in any modern browser. It's a pure frontend project with no dependencies.
2. **Login**: Press **Enter** on the login screen to enter.
3. **General Operation**:
   - Click Dock icons to open apps.
   - Double-click window title bars to maximize.
   - Click the emblem/sun icon in the top-left for the system menu.
   - Right-click desktop for the context menu.
   - Click the language icon in the Dock to switch languages (**FA** -> **EN** -> **ZH**).
4. ⚡ **Trigger Hidden Ending (Liberation)**:
   - Open the **Terminal** app.
   - Enter and execute the command: `override`

### 🛠️ Tech Stack

- **HTML5 / CSS3**: Semantic structure and smooth transitions (Glassmorphism, Depth of Field).
- **JavaScript (Vanilla)**: Window management, terminal logic, and multilingual support without heavy frameworks.
- **Tailwind CSS**: Rapid utility-based styling via CDN.
- **Vazirmatn Font**: Optimized for Persian/Arabic typography.

### 🕊️ Principles & Tribute

This project is a digital protest statement. It pays tribute to:

- **Mahsa (Jina) Amini** and all who sacrificed in the "Woman, Life, Freedom" movement.
- The victims of flight **PS752**.
- Political prisoners of the **1988 massacre** and subsequent protests.

> "Information is liberation. Truth demolishes prisons."

## 中文 (Chinese - ZH)

这是一个基于 Web 的交互式操作系统模拟器与数字艺术项目。它生动地展示了从极权的“国家信息网（SHOMA）”向自由、世俗、民主政权的转变。

### 📖 项目简介

本项目最初呈现为 **Melli OS** (سیستم‌عامل ملی / 国家操作系统) —— 一个受伊朗极权审查与监控启发而设计的虚构操作系统。在这个阶段，用户体验到的是断网、严密的 FATA（网络警察）监控、强制头巾法令以及无处不在的压迫感。

然而，通过在**终端** (Terminal) 中输入特定的“解放协议”命令，用户可以打破国家信息网的防火墙，将系统“黑客入侵”并转化为一个象征光明与自由的新系统。解放后的系统以 "**Woman, Life, Freedom**" (زن، زندگی، آزادی / 女性，生命，自由) 为核心，揭露历史真相，并展示《世俗民主宣言》。

### ✨ 核心功能

- 💻 **沉浸式桌面体验**: 拥有完整的启动动画、登录界面、Dock 栏、可拖拽的拟物化窗口系统。
- 🌍 **多语言与 RTL 支持**: 原生支持 **波斯语 (FA)**、**英语 (EN)** 和 **中文 (ZH)**。波斯语模式下完美适配从右到左 (Right-to-Left) 的阅读排版，并内置开源字体 Vazirmatn。
- 👁️ **网络监控模拟器 (FATA Monitor)**: 实时模拟极权网络监控、数据包拦截、关键字审查和 CPU 负载监控。
- ⌨️ **交互式终端 (Terminal)**: 包含真实的命令行交互（如 `help`, `ls`, `cat`, `whoami`, `history` 等）。
- 🔓 **"解放" 协议**: 通过隐藏指令触发全屏过渡动画，破解防火墙，系统主题从压抑的深绿色转变为象征曙光的三色旗（绿、白、红）色调。
- 📜 **动态历史档案**:
  - **极权状态下**: 显示《革命卫队网络指令》。
  - **解放状态下**: 解锁《女性，生命，自由 - 世俗民主宣言》以及《伊朗神权政府控诉书》。

### 🚀 如何使用

1. **运行项目**: 打开 `index.html` 即可运行，无需构建或安装后端依赖。
2. **登录系统**: 在登录界面，直接按下 **Enter** 键即可进入系统。
3. **常规操作**:
   - 点击底部 Dock 栏图标打开不同应用。
   - 双击窗口顶部标题栏可以最大化窗口。
   - 点击左上角太阳图标可打开系统菜单。
   - 桌面右键可呼出上下文菜单。
   - 在 Dock 栏点击语言图标切换界面语言 (**FA** -> **EN** -> **ZH**)。
4. ⚡ **触发隐藏结局 (解放系统)**:
   - 打开 **终端** (Terminal) 应用。
   - 输入并执行命令：`override`

### 🛠️ 技术栈

- **HTML5 / CSS3**: 语义化结构与流畅的 CSS 过渡动画（毛玻璃效果、景深渐变）。
- **JavaScript (Vanilla)**: 纯原生 JS 实现窗口管理、终端逻辑与多语言切换，零第三方框架。
- **Tailwind CSS**: 用于快速构建响应式原子类样式。
- **Vazirmatn Font**: 专为波斯语/阿拉伯语排版优化的开源字体。
